# Checklist

## GitHub 仓库整理
- [x] GitHub CLI 已安装并可正常使用
- [x] 所有 labels 已同步到 GitHub 仓库
- [x] Labels 颜色、描述与 labels.yml 一致

## Git 管理
- [x] 当前分支已推送到远程仓库
- [x] Commit messages 符合 Conventional Commits 规范
- [x] 无未提交的变更

## Pull Request
- [x] PR 已创建并指向 master 分支
- [x] PR 标题符合 Conventional Commits 格式
- [x] PR 描述包含完整的变更说明
- [x] PR 描述包含验证结果
- [x] PR 描述包含影响范围说明
- [x] PR 关联了相关的 Issue（如有）

## 验证
- [x] 所有 CI 检查通过（或至少 CI workflow 已触发）
- [x] PR 状态为 open 且可合并
